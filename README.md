# soltest

## Get the source code

```
git clone git@github.com:aarlt/soltest.git
```

## Prerequisites

### Ubuntu 17.10

```
sudo apt install build-essential cmake qtbase5-dev qtdeclarative5-dev libpoco-dev libboost-all-dev libleveldb-dev
```

### MacOS

To do.

## Build

After checking out the `soltest` repository.

```
cd soltest
mkdir build
cd build
cmake ..
make
```
